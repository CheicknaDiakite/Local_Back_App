API Backend Entreprise

Bienvenue sur l'API backend de gestion d'entreprise. Cette API fournit des endpoints pour gérer les entreprises, utilisateurs, catégories, entrées, sorties, dépenses et factures.

Endpoints API disponibles :

  • /api/entreprise/ - Gestion des entreprises
  • /api/utilisateur/ - Gestion des utilisateurs
  • /admin/ - Interface d'administration Django

Pour accéder à l'application frontend, veuillez vous rendre sur l'interface appropriée.
